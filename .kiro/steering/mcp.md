# MCP Tool Usage Guidelines

## Component Search Tool

When using the MCP component inspiration tool (`mcp_21st_devmagic_21st_magic_component_inspiration`):

- Only provide the required parameters: `message` and `searchQuery`
- Do not include optional or additional parameters beyond these two
- Keep `searchQuery` concise (2-4 words maximum like hero , background like generic UI component names) (not team hero , home hero) to match UI component patterns
- Use the full user message in the `message` parameter for context
